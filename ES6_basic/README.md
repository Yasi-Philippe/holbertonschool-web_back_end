# ES6 Basic

A collection of JavaScript exercises covering the core ES6 (ECMAScript 2015) features introduced in modern JavaScript. Each task isolates one specific syntax improvement or language feature.

## Requirements

- Node.js with Babel (`@babel/core`, `@babel/node`, `@babel/preset-env`)
- Jest for testing
- ESLint with Airbnb base config

## Setup

```bash
npm install
```

## Running tasks

```bash
npm run dev <file>
# Example:
npm run dev 1-main.js
```

## Running tests

```bash
npm test
```

## Files

| File | Topic |
|------|-------|
| `0-constants.js` | `const` and `let` declarations |
| `1-block-scoped.js` | Block scope with `let` |
| `2-arrow.js` | Arrow function syntax |
| `3-default-parameter.js` | Default parameter values |
| `4-rest-parameter.js` | Rest parameter syntax (`...args`) |
| `5-spread-operator.js` | Spread syntax (`...`) |
| `6-string-interpolation.js` | Template literals |
| `7-getBudgetObject.js` | Object property shorthand |
| `8-getBudgetCurrentYear.js` | Computed property names |
| `9-getFullBudget.js` | ES6 method shorthand |
| `10-loops.js` | `for...of` loop |
| `11-createEmployeesObject.js` | Dynamic object keys |
| `12-createReportObject.js` | Spread in objects + method shorthand |
